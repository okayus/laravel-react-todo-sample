# Product Overview

技術学習を目的としたシンプルなTODOアプリケーション。
Laravel + React + Inertia.js のモダンなフルスタック構成を実践的に習得するための教材プロジェクト。

## Core Capabilities

- TODOの作成・編集・削除・完了管理
- シンプルで直感的なUI
- SPA的なユーザー体験（Inertia.js）

## Target Use Cases

- フルスタック開発の学習
- Laravel + React + Inertia.js の統合パターン習得
- Docker/AWS/CI/CDのデプロイフロー学習

## Value Proposition

- **学習目的特化**: 実用性よりも技術習得を優先
- **モダンスタック**: 現在主流の技術構成を網羅
- **本番環境あり**: ローカル開発からデプロイまで一気通貫

## Constraints

- 個人開発プロジェクト
- 宣伝・実用目的なし
- トラフィックはほぼゼロ想定
